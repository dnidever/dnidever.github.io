Software

APOGEE Reduction Pipeline

  1. Fully-automated pipeline to reduce the raw up-the-ramp APOGEE spectroscopic data consisting of 300 fiber spectra per observation (with Jon Holtzman, in IDL, ~20,000 lines of code).  Publicly available with the SDSS-III Data Release 10 (DR10, and later DRs) and described in Nidever et al. (2015).

PHOTRED

  1. Fully-automated, generic, photometric reduction package based on DAOPHOT/ALLSTAR/ALLFRAME.  Any flat images can be input and final, calibrated, PSF photometry (with accurate astrometry) plus diagnostic webpages with plots are output (in IDL, ~40,000 lines of code).  Used to reduce the photometry for the MAgellanic Periphery Survey (MAPS) and the Survey of the Magellanic Stellar History (SMASH). Publicly available via github and described in Nidever et al. (2017).

SPECFIT

  1. Fully-automated software to fit synthetic spectra to observed spectra in order to obtain stellar parameters Teff, logg, [M/H], [α/Fe], RV and vsini (in IDL, ~7,700 lines of code).

Gaussian Decomposition Software

  1. Fully-automated software to Gaussian decompose spectra in an HI datacube.  (in IDL, ~4,600 lines of code).  Used to decompose the entire Leiden-Argentine-Bonn HI all-sky Survey and study the origin of the Magellanic Stream. Described in Nidever, Majewski & Burton 2008, ApJ, 679, 432.

EZ_SPAM

  1. Easy Stellar Parameters And Metallicites.  Stellar parameters (Teff, logg) and metallicities are obtained for observed stellar spectra using calculated Lick indices and fitting functions (with J.Carlin, in IDL, ~2,000 lines of code).  Briefly described in Carlin, Grillmair, Muñoz, Nidever & Majewski 2009, ApJ, 702, 9L.


GALSAT

  1. N-body (and restricted N-body) code for modeling galactic satellites in the MW potential (based on the N-body code by Piet Hut and Jun Makino).  Includes a program to automatically find the best-matching MW satellite orbit (or MW satellite tidal tails) to observed data - using a genetic algorithm to search the orbital parameter space (in C++ with IDL interface, ~1,800 lines of code).
